#  Module-7 Part 1-Assessment-2 
Calculator App
Features
Digit buttons (0-9): Input numbers into the display
Operators (+, -, ×, ÷): Perform arithmetic operations
AC button: Clear the display and reset the calculator
Equals button (=): Calculate and display the result
How it works
Click digit buttons to enter numbers
Click an operator button to select an operation
Enter the second number
Click equals to see the result
Click AC to reset and start over
Files
calculator.html - Main HTML file with calculator layout and JavaScript logic
style.css - Styling for the calculator interface (referenced but not shown)
Code Structure
The calculator uses these JavaScript functions:

digitBtnPressed() - Handles number input
operatorBtnPressed() - Stores the operator and first value
btnEqualsPressed() - Performs the calculation
btnACPressed() - Clears the display
